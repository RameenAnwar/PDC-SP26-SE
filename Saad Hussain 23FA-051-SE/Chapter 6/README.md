## How to run

From this directory (`Chapter 6`), run each example from its own folder.

Socket time server:

```bash
cd Code/Socket
python server.py
python client.py
```

Socket file transfer:

```bash
cd Code/Socket
python server2.py
python client2.py
```

Pyro4 examples require the Pyro name server to be running first:

```bash
python -m Pyro4.naming
```

Celery examples require a running AMQP broker such as RabbitMQ before submitting tasks.

## Concepts covered

- TCP socket client/server communication.
- Sending text and file data over sockets.
- Remote method calls with Pyro4.
- Registering and locating remote objects with a Pyro name server.
- Chain topology communication between remote objects.
- Asynchronous task submission with Celery.

## Requirements

- Python 3.x
- Python packages: `Pyro4`, `celery`
- RabbitMQ or another AMQP broker for Celery examples
